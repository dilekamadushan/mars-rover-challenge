# mars-rover-challenge
## Steps to start locally

1.  `gradlew build`
2. `java -jar build/libs/MarsChallenge-1.0-SNAPSHOT.jar`.

The program expects input as follows:

```
5 5
1 2 N
LMLMLMLMM
```
3.Type `exit` to quit the program
